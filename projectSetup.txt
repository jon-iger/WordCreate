This project uses the third party API service SerpAPI. For security, contact the developer to receive the key to use this API. Then, follow these steps to set up the project:

1. Open the Xcode project and create a new target with the WordCreate project
2. Go the Edit Schemes menu for the target, and click on Run->Arguments
3. Under the environment variables section, add this variable to the list:
- Key: serpAPIKey, value: insert-API-key-here

API keys should never be distributed without the written authorization of the developer
